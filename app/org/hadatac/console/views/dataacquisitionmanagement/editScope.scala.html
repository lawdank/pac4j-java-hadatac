Select Stream Specification's Scope

The scope of a data stream can be an object collection, like a group of subjects or a collection of samples. Data streams that are related to object collections often include some identifier used to assign values from the files to specific objects in the object collection. The scope of a data stream can also be a single object meaning that the entire content value of the data acquisition is about attributes form the scope object. An example of such king of data acquisition is a complete blood count where all the values of the data acquisition are related to a single subject.
